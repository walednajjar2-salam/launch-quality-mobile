Launch Quality LLC v47 — Railway Production

Repo: https://github.com/walednajjar2-salam/jawdah-cloud-v47

1) Upload ALL files in this folder to GitHub repo root (not inside a subfolder)
2) Railway → New Project → Deploy from GitHub → select jawdah-cloud-v47
3) Variables:
   JAWDAH_HOST=0.0.0.0
   JAWDAH_DATA_DIR=/app/data
4) Networking → Generate Domain
5) Test: https://YOUR-APP.up.railway.app/api/health
6) Set ADMIN_PASSWORD in Railway variables, then login as admin

Optional: add Volume mounted at /app/data for persistent SQLite.

Start command: python server.py
Health check: /api/health
