تثبيت Launch Quality ERP

ثبّت النظام كتطبيق على Windows أو Android، ثم سجّل الدخول بحسابك الوظيفي.

Windows

  1. افتح الرابط في Microsoft Edge أو Google Chrome.
  2. من شريط العنوان اضغط أيقونة التثبيت، أو افتح القائمة واختر تثبيت التطبيق.
  3. سيظهر التطبيق في Start Menu وسطح المكتب حسب إعدادات المتصفح.

Android

  1. افتح الرابط في Chrome.
  2. اضغط القائمة ثم Install app أو Add to Home screen.
  3. ستظهر أيقونة Launch Quality ERP ضمن تطبيقات الهاتف.

للتثبيت خارج جهازك يجب أن يكون الرابط HTTPS عام أو ضمن بيئة آمنة. رابط 127.0.0.1 يعمل على جهاز السيرفر فقط.
