# Bank Home Demo — تعلّم واجهة الهاتف

واجهة تجريبية مستوحاة من شاشة رئيسية بنكية، **لأغراض التعلّم فقط**.  
ليست تطبيقاً بنكياً حقيقياً ولا تتصل بأي بنك أو حساب.

## تعديل الرصيد

الطريقة الأسهل: من داخل الشاشة نفسها  
اكتب الرقم في خانة **«تعديل الرصيد»** ثم اضغط **«تطبيق الرصيد»**.

أو عدّل القيمة الافتراضية في الكود:

1. **HTML** — `learning/bank-home-demo.html`

   ```js
   balance: 7034.340,
   ```

2. **Flutter** — `lib/learning/bank_home_demo_data.dart`

   ```dart
   static const double balance = 7034.340;
   ```

   التشغيل:

   ```bash
   flutter run -t lib/main_bank_demo.dart
   ```

## ماذا تتعلّم من هذا المثال؟

- فصل البيانات عن الواجهة (`bank_home_demo_data.dart` / `DEMO_DATA`)
- بناء بطاقة رصيد، أزرار سريعة، قائمة معاملات، وشريط تنقّل سفلي
- إخفاء/إظهار الرصيد بزر العين
- تنسيق العملة بثلاث خانات عشرية (OMR)
