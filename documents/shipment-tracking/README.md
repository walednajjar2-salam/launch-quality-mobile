# Shipment Tracking — Land Rover Defender → Sohar Port

لوحة تتبع شحنة المركبة بأسلوب K LINE مع:

- صورة حقيقية 100% لـ Land Rover Defender
- خريطة بشكل جديد (إطار مقصوص + مسار حي عبر Leaflet)
- بيانات العميل والتأمين (الخليج العربي للتأمين)

## التشغيل

افتح `index.html` في المتصفح، أو:

```bash
python3 -m http.server 8770 --directory documents/shipment-tracking
```
